# Bun for Front-end application

## How to create a project

### Create a React app

```bash
$ bun create vite <project-name> -- template react-swc-ts
$ cd <project-name>
$ bun install
$ bun run dev
$ bun run build
```

#### Template can choose

- vanilla
- vanilla-ts
- vue
- vue-ts
- react
- react-ts
- react-swc
- react-swc-ts
- preact
- preact-ts
- lit
- lit-ts
- svelte
- svelte-ts
- solid
- solid-ts
- qwik
- qwik-ts
